API de la Aplicación UAM Departamento de Física
===============================================

Esta es la documentación de la API de la aplicación de la UAM Departamento de Física.

.. important:: 
   
      La documentación está en proceso de construcción.

Necesidades de la aplicación
============================

1. Tener instalado Python 3.7 o superior.
2. Tener instalado pip.
3. Tener instalado el paquete de Sphinx.
4. Tener instalado MySQL.

Instalación
===========

Para instalar la aplicación, se debe clonar el repositorio de la aplicación y ejecutar el siguiente comando:

.. code-block:: bash

   $ pip install -r requirements.txt

Uso
===

Para usar la aplicación, se debe ejecutar el siguiente comando desde la raíz del proyecto:

.. code-block:: bash

   $ python -m src
